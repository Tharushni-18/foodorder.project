 🍔 Online Food Ordering System

 📌 Project Description

The **Online Food Ordering System** is a console-based Java application that allows users to select food items from a menu, enter quantity, and generate the total bill.
This project demonstrates core Java concepts like loops, switch-case, user input handling, and basic arithmetic operations.


 🚀 Features

* 📋 Display food menu
* 🛒 Add items to cart
* 🔢 Enter quantity
* 💰 Calculate total bill
* ❌ Exit option
* 🧾 Final bill display


🛠️ Technologies Used

* Java
* Scanner Class
* Conditional Statements
* Loops
* Switch Case

 ▶️ How to Run the Project

Step 1: Compile the Program

javac FoodOrderingSystem.java

 Step 2: Run the Program

java FoodOrderingSystem
